## 安装使用

- 获取项目代码

```bash
git clone https://github.com/zeke202207/netx-app.git
```

- 安装依赖

```bash
cd netx-app

pnpm install

```

- 运行

```bash
pnpm serve
```

- 打包

```bash
pnpm build
```
